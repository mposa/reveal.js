A (Controlled) Hybrid Systems View

Polcy: controller $u(x)$, hybrid guard $G(x)$, and discrete controller $s(x)$ where

  • Execute $u(x)$ whenever $G(x) > 0$
  • Take a step with discrete input $s(x)$ whenever $G(x) = 0$

Lyapunov Functions

  • $G(x) > 0 \Rightarrow \dot V(x,u(x)) \leq 0$
  • $G(x) = 0 \Rightarrow V(R(x,s(x))) \leq V(x)$

But $G,u,s$ are all unknown, so this is non-convex$^*$
